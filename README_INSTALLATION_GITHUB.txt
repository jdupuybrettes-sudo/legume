GESTION FRUITS & LEGUMES - VERSION GITHUB PAGES

Objectif :
Mettre l'application HTML en ligne sur GitHub Pages pour obtenir une adresse web utilisable sur iPhone, Android et PC.

Fichiers du dossier :
- index.html : application complète.
- .nojekyll : fichier technique pour GitHub Pages.
- README_INSTALLATION_GITHUB.txt : procédure.

PROCEDURE SIMPLE SANS LOGICIEL :

1. Aller sur https://github.com
2. Créer un compte ou se connecter.
3. Cliquer sur + puis New repository.
4. Nom du dépôt conseillé : gestion-fruits-legumes
5. Choisir Public pour une publication gratuite simple.
6. Cliquer sur Create repository.
7. Dans le dépôt, cliquer sur Add file puis Upload files.
8. Glisser les fichiers index.html et .nojekyll dans la page.
9. Cliquer sur Commit changes.
10. Aller dans Settings.
11. Dans le menu de gauche, cliquer sur Pages.
12. Dans Build and deployment :
    - Source : Deploy from a branch
    - Branch : main
    - Folder : /root
13. Cliquer sur Save.
14. Attendre quelques minutes.
15. L'adresse sera du type :
    https://NOMUTILISATEUR.github.io/gestion-fruits-legumes/

UTILISATION IPHONE :
1. Ouvrir l'adresse GitHub Pages dans Safari.
2. Cliquer sur Partager.
3. Cliquer sur Ajouter à l'écran d'accueil.
4. L'application apparaîtra comme une icône.

IMPORTANT :
- Les données sont enregistrées dans le navigateur de chaque appareil.
- Si 3 personnes utilisent l'application, elles n'auront pas automatiquement la même base.
- Pour sécuriser les données, utiliser régulièrement Exporter sauvegarde JSON.
- Pour reprendre des données sur un autre appareil, utiliser Importer JSON.

Mise à jour de l'application :
Pour remplacer la version en ligne, retourner dans le dépôt GitHub, supprimer/remplacer index.html, puis Commit changes.
